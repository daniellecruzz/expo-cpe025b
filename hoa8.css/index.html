01 — Overview

About Sass

Sass is a stylesheet language that compiles to CSS. It gives you access to powerful tools that make CSS more maintainable, readable, and fun to write — especially at scale.

02 — Features

What You Get

  • $
    Variables

    Store colors, fonts, and values in reusable tokens.

  • @
    Mixins

    Define reusable chunks of styles, with optional arguments.

  • {}
    Nesting

    Write selectors that mirror your HTML structure cleanly.

03 — Example

See It in Action

$primary: #2563eb;

@mixin flex-center {
  display: flex;
  align-items: center;
  justify-content: center;
}

.hero {
  @include flex-center;
  background: $primary;

  h1 { color: white; }
}
